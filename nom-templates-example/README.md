# Nom template examples

These are the original/default `.nom` templates. Copy them into your repo's `.nom/` folder to customize AI instructions for your feed:

- `pull_request_instructions.md` → `.nom/pull_request_instructions.md`
- `push_instructions.md` → `.nom/push_instructions.md`
- `release_instructions.md` → `.nom/release_instructions.md`

Edit the copies in `.nom/` to define your own voice and posting criteria.
